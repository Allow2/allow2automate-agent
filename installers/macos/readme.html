Configuration File Required

This installer requires a configuration file: allow2automate-agent-config.json

Please ensure the configuration file is in the same folder as this installer package.

You can download the configuration file from the Allow2 Automate parent application.

Installation Details

Files to be installed:

  • Service: /usr/local/bin/allow2automate-agent
  • Helper: /usr/local/bin/allow2automate-agent-helper
  • Config: /Library/Application Support/Allow2/agent/config.json
  • Logs: /var/log/allow2automate-agent.log

Services:

  • LaunchDaemon: /Library/LaunchDaemons/com.allow2.automate-agent.plist
  • LaunchAgent: /Library/LaunchAgents/com.allow2.agent-helper.plist

After Installation

The Allow2 Automate Agent will:

  1. Start automatically on system boot
  2. Connect to your parent server
  3. Begin monitoring and enforcing policies
  4. Display status in menu bar (via helper app)

You can modify the configuration after installation through the Allow2 Automate parent application.
