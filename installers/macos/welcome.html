Welcome to Allow2Automate Agent

This installer will install the Allow2Automate Agent on your Mac.

The agent will:

  • Install system service to /usr/local/bin/
  • Create LaunchDaemon for automatic startup
  • Install helper application for user notifications
  • Connect to your Allow2Automate parent server

Requirements:

  • macOS 10.15 (Catalina) or later
  • Administrator privileges

Click Continue to proceed with installation.
